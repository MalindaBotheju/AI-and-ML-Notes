# 🧠 AI & Machine Learning Study Vault

Welcome to my personal repository of study notes, cheat sheets, and guides covering core concepts in Artificial Intelligence, Machine Learning, and Data Science. 

The goal of this repository is to break down complex mathematical and algorithmic concepts into simple, highly visual, and easy-to-understand documents.

## 📂 What's Inside

### 🤖 1. Machine Learning (Current Focus)
This section contains foundational ML algorithms, optimization techniques, and mathematical intuition.

**Featured Notes:**
* **`Linear Regression.pdf`**: A comprehensive, step-by-step guide covering:
  * Supervised vs. Unsupervised Learning basics.
  * The math behind the "Line of Best Fit" ($y = wx + b$).
  * **Gradient Descent**: How models learn by minimizing errors (including 3D cost function landscapes).
  * **Regularization**: Preventing overfitting using **Ridge (L2)**, **Lasso (L1)**, and **Elastic Net** with a clear comparative breakdown.

## 📝 Format and Style
The notes in this repository are formatted to be visually balanced and intuitive. Rather than just raw code, you will find:
* **Clean Mathematical Equations** explaining what happens under the hood.
* **Visual Graphs & Analogies** (like the "Blindfolded Hiker" for Gradient Descent).
* **Q&A Breakdowns** summarizing *why* we use specific techniques in the real world.
